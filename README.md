# STALKERText

## How to run?

- Clone the repo
- CD to repo folder
- Run
```powershell
lua stalkertext.lua
```

## What is STALKERText?

Inspired by the book Roadside Picnic and the subsequent Movies & Game spinoffs, STALKERText is a text-based RPG written in LUA.

## Why STALKERText?

I like text-based RPGs.

## What is the game loop?

- Enter the ZONE
- Find a new room
- Fight enemies or find LOOT
- Rinse & repeat!
